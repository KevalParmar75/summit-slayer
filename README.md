## ğŸš€ Getting Started

Follow these instructions to get a local copy of Talent Tuner up and running on your machine.

### Prerequisites

You must have **Anaconda** installed to manage the virtual environment properly. 

### Installation

1. **Clone the repository**
   ```bash
   git clone [https://github.com/KevalParmar75/talent-tuner.git](https://github.com/KevalParmar75/talent-tuner.git)
   cd talent-tuner
   conda create --name talent-tuner-env python=3.9
   conda activate talent-tuner-env
   pip install -r requirements.txt
   GEMINI_API_KEY=your_api_key_here
   ```
